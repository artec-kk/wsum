# wsum
Scartch3.0でメッセージとそれを使用しているスプライトを調査するコマンド

## Usage

wsum.exe sb3_project [csv]

Scratch3.0のプロジェクトを渡すと、メッセージとそれを使用しているスプライトのリストを下記のjson形式で出力します。  
>{'変数名1': ['スプライト1', 'スプライト2'...], '変数名2': ['スプライト1', 'スプライト3'...] ... }

csvオプションを付けるとcsv形式で出力します。  
>変数名1, スプライト1, スプライト2  
変数名2, スプライト1, スプライト3  
...
